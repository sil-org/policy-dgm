# Introduction

This Data Governance Policy[^1] establishes the framework for managing data assets within SIL Global. Effective data governance ensures that our data is accurate, secure, accessible, and used legally and ethically, supporting our mission, programs, and stakeholders. This policy will be reviewed annually.

[^1]: Github source for this [document: https://github.com/sil-org/policy-dgm](https://github.com/sil-org/policy-dgm)
